# Get started with the project

### Clone the repository

```sh
git clone <url>
```

## Open two terminals for backend & frontend

### For backend

```sh
cd backend
mkdir -p uploads/thumbnails
pnpm install
pnpm dev
```

> If git didn't track uploads/thumbnails

### For frontend

```sh
cd frontend
pnpm install
pnpm dev
```

**Note: .env.example has specified environment variables for both backend & frontend**
